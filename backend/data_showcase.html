股票信息采集系统 - 实时数据展示

🚀 系统状态

后端服务：http://localhost:8000 运行中

前端服务：http://localhost:3002 运行中

数据源：Twelve Data API 已连接

样本股票：10只纳斯达克股票（AAPL, MSFT, NVDA, META, NFLX, PYPL, INTC, CSCO, ADBE, QCOM）

📊 核心功能展示

  • 个性化排名系统 - 基于用户优先级的多维度股票排名
  • 活跃度排名 - 基于成交量和成交额（权重30%）
  • 波动性排名 - 基于日内振幅（权重25%）
  • 涨跌排名 - 基于当日表现（权重20%）
  • 市值排名 - 基于公司市值（权重15%）
  • 价格排名 - 基于股票价格（权重10%）
  • 综合排名 - 多维度加权算法，符合用户个性化需求

⚙️ 技术架构

  • 前端：React 18 + TypeScript + Ant Design 5 + Vite
  • 后端：FastAPI + SQLAlchemy + SQLite
  • 数据源：Twelve Data API (实时市场数据)
  • 认证：JWT Token Based Authentication
  • API文档：OpenAPI/Swagger 自动生成
